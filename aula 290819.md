## Implementação de Bibliotecas e Headers

O código principal fica assim:

```
#include <stdlib.h>
#include "data.h"

tData lerData() {
  ...
}

void apresentaData() {
  ...
}
```


Enquanto o Header fica assim:
```
#ifndef _DATA
  #define _DATA

  typedef struct Data {
    int dia, mes, ano;
  } tData;
  tData lerData();
  void apresentaData();
  ...
#endif
```
